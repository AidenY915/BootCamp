다음 용어들에 대해 서술하시오.

1. DBMS?
DBMS(Database Management system)란 데이터 베이스 관리 시스템으로서, 데이터베이스를 운영하고 관리하는 소프트웨어다. 주로 관계형 DBMS(RDBMS)를 많이 쓴다. RDBMS에서는 SQL을 통해 데이터베이스를 조작한다.
2. DDL이란? data definition language로서, 테이블 같은 데이터 구조를 정의하는 명령어들을 의미한다. 컬럼의 생성 삭제 변경 등에 사용한다.
3. DML이란?
data manipulate language로서, insert select delete update 등의 기능을 구현한 언어를 의미한다.
튜플의 생성, 삭제, 수정등에 이용된다.
4. DCL이란?
data control language로서, 데이터베이스나 테이블로의 접근 권한을 관리하는 명령어이다.


5. 트랜잭션이란? 논리적인 단위의 작업을 의미하며, 하나의 트랜잭션이 실행 중 에러가 발생한다면, 트랜잭션의 시작 직전으로 롤백하여 재실행 하고. 하나의 트랜잭션이 에러 없이 완료된 시점에 커밋한다.
6. RDBMS 란? 관계형 데이터베이스로서, 테이블의 형태를 유지하며
7. CRUD 란? 
기본적인 데이터 처리기능인 Creat Read Update Delete 이렇게 4가지 기능을 의미한다. 
8. 튜플(Tuple)란?
튜플이란, 테이블에서의 하나의 행을 의미하며, 서로 연관된 속성값들의 집합을 뜻한다.
9. 릴레이션? 관계형 데이터베이스에서 테이블 객체를 의미한다.
10. 도메인? 하나의 속성이 가질 수 있는 값의 범위를 의미한다.
11. 차수(Degree)? 속성의 수를 의미한다.
12. 카디날리티(Cardinality)? 튜플, 즉, 행의 수를 의미한다.
